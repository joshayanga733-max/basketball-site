# Installation Guide

## Setup Instructions
1. Clone the repository:
   
   ```bash
   git clone https://github.com/joshayanga733-max/basketball-site.git
   ```

2. Change into the directory:

   ```bash
   cd basketball-site
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

## How to Run the Site Locally
1. Start the development server:
   
   ```bash
   npm start
   ```

2. Open your browser and go to `http://localhost:3000` to view the site.

## System Requirements
- Node.js version 14 or higher
- npm (Node Package Manager)
- Any modern web browser (Chrome, Firefox, Safari, etc.)

### Additional Notes
- Ensure you have Node.js and npm installed on your machine. To check if these are installed, run:

   ```bash
   node -v
   npm -v
   ```

- If you encounter any issues, refer to the project's README file for more information on troubleshooting.
